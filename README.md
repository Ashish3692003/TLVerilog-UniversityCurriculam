# TLVerilog-examples

A community collection of TL-Verilog (TLV) circuit examples for the **curricula-summer-2026** program, maintained by the TL-Verilog Users community.

## About

This repository serves as a shared resource where contributors can add, reference, and build upon circuit examples using TL-Verilog and related tools. The goal is to support the development of summer 2026 curriculum content.

## Repository Structure

```
/digital/         - Digital circuit examples (logic gates, flip-flops, counters, etc.)
/analog/          - Analog circuit examples (op-amps, filters, etc.)
/tlv-converted/   - Examples converted to TL-Verilog (.tlv files)
```

## Tools & Resources

| Tool | Link | Description |
|------|------|-------------|
| EDA Playground | https://www.edaplayground.com | Online Verilog/VHDL/SystemVerilog simulator |
| CircuitVerse | https://circuitverse.org | Digital logic circuit simulator |
| EveryCircuit | https://everycircuit.com | Animated analog & digital circuit simulator |
| Tinkercad Circuits | https://www.tinkercad.com/circuits | 3D breadboard-style visual simulator |
| DCACLab | https://dcaclab.com | Real-time component interplay simulator |
| CircuitLab | https://www.circuitlab.com | Schematic editor with in-browser simulation |
| Multisim | https://www.multisim.com | Industry-standard education circuit simulator |

## Licensing
